## Assembly Steps (all in Kling)

1. Generate each clip one at a time (9 total)
2. Review each — regenerate any that don't look right (expect 2-3 attempts per clip)
3. Use Kling's **Multi-Shot / Custom Multi-Shot** feature to chain clips together into one video
4. Add text overlays using Kling's built-in text editor (white text, clean sans-serif font, center-bottom or center placement)
5. Export the final video

---

## Tips

- **Generate 2-3 versions** of each clip and pick the best
- **Keep the same character description** across all technician clips (navy uniform, friendly face) for visual consistency
- If a clip has weird hands or distorted faces, regenerate — don't settle
- For the logo clip, you can upload your `applogo.png` as a reference/start frame
- The text overlays should use white text with a very subtle dark shadow for readability
- Keep text on screen for the full 5 seconds — viewers need time to read
